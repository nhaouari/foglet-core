    StreamRequest

    Extends:

    * → StreamRequest

    A StreamRequest enable to stream data to a peer using a push-based API.

    Example:

    const foglet = getSomeFoglet();
const peerID = getSomePeerID();

const stream = foglet.streamUnicast(peerID);
stream.write('Hello');
stream.write(' world!');
stream.end();
